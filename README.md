# 💄 Cosmetic Insights: Navigating Cosmetics Trends and Consumer Behavior with Tableau

## 📌 Project Overview

This project analyzes cosmetic product data using Tableau to identify market trends, consumer preferences, pricing patterns, and product performance. The dashboard helps businesses understand customer behavior and make data-driven decisions.

---

## 🎯 Objectives

- Analyze cosmetic products using Tableau.
- Compare brands and product categories.
- Study product pricing trends.
- Analyze customer ratings and reviews.
- Identify skin-type suitability.
- Build interactive dashboards and stories.
- Generate business insights for decision-making.

---

## 📂 Dataset Information

The dataset contains the following fields:

- Product Name
- Brand
- Category
- Price
- Rank
- Rating
- Reviews
- Ingredients
- Skin Type

## 📊 Project Workflow

### 1. Data Collection
- Collected cosmetic product dataset.
- Verified data quality.

### 2. Data Preparation
- Removed duplicate records.
- Handled missing values.
- Corrected data types.

### 3. Data Visualization
Created multiple Tableau visualizations including:
- Brand-wise Product Count
- Category Distribution
- Price Analysis
- Rating Analysis
- Skin Type Analysis

### 4. Dashboard
Designed an interactive dashboard with filters for:
- Brand
- Category
- Skin Type

### 5. Story
Created Tableau Story explaining:
- Market Trends
- Consumer Preferences
- Product Performance

### 6. Web Integration
Integrated the Tableau dashboard using Flask.

---

## 📈 Key Insights

- Popular cosmetic brands were identified.
- Most preferred product categories were analyzed.
- Pricing trends across brands were compared.
- Consumer ratings helped identify top-performing products.
- Skin-type suitability analysis supports personalized recommendations.

## 🚀 How to Run

1. Download the repository.
2. Install Python dependencies.

```
pip install -r requirements.txt
```

3. Run the Flask application.

```
python app.py
```

4. Open the browser.

```
http://127.0.0.1:5000
```
