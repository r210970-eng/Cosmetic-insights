## 📊 Dataset

Key fields used:

| Field | Description |
|---|---|
| Brand | Cosmetics brand name |
| Product Category | e.g., Moisturizer, Cleanser, Serum |
| Price | Product price |
| Rank | Product ranking/rating |
| Skin Type | Dry, Oily, Combination, Normal, Sensitive |

---

## 📈 Visualizations

The dashboard includes 6 unique visualizations:

1. Bar Chart — Product count by brand/category
2. Box Plot / Histogram — Price distribution by category
3. Heat Map — Brand vs. skin-type coverage
4. Scatter Plot — Price vs. Rank (value-for-money)
5. Treemap — Market share by brand
6. Pie/Bar Chart — Skin-type suitability breakdown

An interactive **Dashboard** (filters: Brand, Category, Skin Type, Price Range) and a 3-scene **Story** (one scene per business scenario) are built on top of these.
