# Cosmetic Insights: Navigating Cosmetics Trends and Consumer Behavior with Tableau

## Project Overview
This project analyzes cosmetic product data using Tableau to identify market trends, consumer preferences, product performance, pricing patterns, and skin type suitability.

## Objectives
- Analyze cosmetic products using Tableau.
- Compare brands and product categories.
- Study pricing trends.
- Analyze customer ratings and reviews.
- Build an interactive dashboard and story.

## Dataset
The dataset contains:
- Product Name
- Brand
- Category
- Price
- Rank
- Skin Type
- Rating
- Reviews
- Ingredients

## Tools Used
- Tableau Public
- Microsoft Excel
- Python
- Flask
- GitHub

## Project Features
- Data Preparation
- Interactive Visualizations
- Dashboard

- Business Insights
